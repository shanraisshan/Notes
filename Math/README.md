# PERCENTAGE
```
% = 1/100
```

### 50% of 80
```
result = 50% of 80 
result = (50/100) x 80
result = 40
```

### 40 is What% of 80
```
40 = result% of 80 
40 = (result/100) x 80
(40 x 100)/80 = result
50 = result
```

### Increase from 100 to 140, find % increase.
```
140-100 = 40          ➼ [incr-actual = diff]
(40/100)x100 = 40%    ➼ [diff/actual) x 100 = incr%]
```
